## Add as a Dependency

### SBT
```
"com.nthportal" %% "extra-predef" % "1.1.1"
```

### Maven

### Scala 2.12
```
<dependency>
  <groupId>com.nthportal</groupId>
  <artifactId>extra-predef_2.12</artifactId>
  <version>1.1.1</version>
</dependency>
```
